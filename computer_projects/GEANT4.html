GEANT4


The GEANT4 section of this website is intended to give a relatively detailed tutorial on how to install and use GEANT4 from the point of initial installation on a blank Ubuntu box, through to building a Cherenkov detector. Each section of this process is detailed in a seperate page. The source code used throughout this process can be accessed via GitHub.
