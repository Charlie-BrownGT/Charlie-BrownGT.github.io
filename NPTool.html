NPTool


NPTool (Nuclear Physics Tool) is a simulation tool set used in low energy nuclear physics to model a variety of nuclear reactions in different detector setups. It can be used to simulate results prior to physical experiments being carried out.


NPTool relies on several other tools and packages in order to function, there is an installation guide provided in the following which details the setup of NPTool from a clean virtual machine, through to testing the functionality of the tool.
